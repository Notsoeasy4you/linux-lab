#Linux Basic Commands

## pwd
Print working directory.

- Example:
pwd
___

## ls
List working directory

- Examples:
ls
ls -l
ls -la
___

## cd
Change directory

- Example:
cd /home
cd ..
cd ___

## mkdir
Create new directory.

- Example:
mkdir demofolder
___

## rm
Remove files.

- Example:
rm file.txt
rm -r folder
